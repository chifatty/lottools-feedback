## English

This repository is the public feedback portal for Lottools.

- Live site: https://lottools.cc
- [Create an issue](https://github.com/chifatty/lottools-feedback/issues/new)

Use this repository to:
- report bugs
- suggest improvements
- share feature requests
- send general feedback

You may write your issue in any language.

Suggested details to include:
- page URL
- what happened
- steps to reproduce
- screenshot (optional)
- browser / device (optional)

Notes:
- The main Lottools source repository is private.
- Please use Issues for feedback.
- Please do not open pull requests in this repository.
